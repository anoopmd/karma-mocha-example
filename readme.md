# Karma Mocha Example

A Simple Karma Mocha Example

## Install

```sh
$ yarn install
$ bower install
```

## Launch Karma
```sh
$ node_modules/karma/bin/karma start
```

## Run Jasmine in browser
```sh
open index.spec.html in your browser
```
